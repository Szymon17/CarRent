# Car Rental App

## Overview
Car Rental App is a web application that allows users to browse available cars and make online reservations. The system is split into two parts: a customer-facing shop and an admin panel for managing the platform.

## Features
- Browse and filter available cars
- View detailed car information
- Make and manage reservations
- User authentication system
- Responsive UI for mobile and desktop

## Tech Stack
- React
- TypeScript
- Node.js
- REST API

## How to install
- You will need your own MongoDB database
- Clone this repository
- Install dependencies
- Create a `.env` file in the server folder and set variables:
  - HOST
  - USER
  - DB_PORT
  - DB_PASSWORD
  - DB_NAME
  - PORT
  - SECRET_KEY
- Run the application
- Enjoy
